# GS373 — Coding Exercises

This repo contains ungraded coding exercises for GS373. The goal of these exercises is to give you a low-stakes space to play around with code, break things, and build intuition.

## How to use this repo

Each week has its own folder. Inside you will find one or more notebooks to work through at your own pace.

## Opening notebooks in Google Colab

If you don't have your conda envoroment set upyou can do some of these exercises using google colab.

1. Find the notebook you want to open in the repo
2. Copy the URL from your browser
3. Replace `github.com` with `githubtocolab.com` in the URL and hit enter
4. The notebook will open directly in Google Colab — no setup required!
